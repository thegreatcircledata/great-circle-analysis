## Quick Wins (likely to produce clear results fast)
- **03 Geomagnetic** — mostly computational, data is freely downloadable, result is almost certainly null but adds rigor
- **04 Temporal Propagation** — uses existing data from previous analyses, wavefront plot alone is worth making
- **02 Astronomical** — computationally intensive but data is all calculable (no external databases needed)

## Highest Upside (could produce breakthrough findings)
- **01 Early Holocene** — characterizing what the corridor was BEFORE monuments could reframe the entire narrative
- **08 Ahramat Branch** — the perpendicular intersection probability could be the single most striking statistic in the project
- **10D Predynastic** — if the pattern predates pyramids, everything changes

## Most Exploratory (may or may not pan out)
- **05 Oral Tradition** — data access is the bottleneck; if Berezkin's database is accessible, this could be extraordinary
- **07 Songlines** — culturally important and methodologically novel; requires careful handling
- **09 Polynesian** — connects the Pacific segment to navigation history; data is scattered

## Suggested Execution Strategy
1. Run 03, 04, 02 first (quick wins, mostly computational)
2. Run 08, 10 next (Egypt-focused, builds on strongest existing signal)
3. Run 01, 06 (medium data difficulty, high interpretive value)
4. Run 05, 07, 09 last (highest data acquisition effort)
